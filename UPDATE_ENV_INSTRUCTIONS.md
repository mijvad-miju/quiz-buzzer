# Update Environment Variables

Replace the content of your `.env` file with:

```env
VITE_SUPABASE_PROJECT_ID="YOUR_NEW_PROJECT_ID"
VITE_SUPABASE_PUBLISHABLE_KEY="YOUR_NEW_ANON_KEY"
VITE_SUPABASE_URL="https://YOUR_NEW_PROJECT_ID.supabase.co"
```

Replace:
- `YOUR_NEW_PROJECT_ID` with your actual project ID
- `YOUR_NEW_ANON_KEY` with your actual anon key
- `YOUR_NEW_PROJECT_ID` in the URL with your actual project ID
